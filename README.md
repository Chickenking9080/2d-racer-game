# Extreme Racer
A top-down car racing prototype game built in Godot. This project features custom pixel art and a modular car controller adapted from my previous work.

## Controls
* **W:** Accelerate
* **S:** Brake/Reverse
* **A/D:** Steer
* **Space:** Drift

## Features
* **Custom Sprites:** All environment and vehicle assets were made using **Piskel** a online pixel art software.
* **Smooth Acceleration** Uses a acceleration system to make acceleration and speed smooth
* **Lap System (In-Progress):** The backend logic for checkpoints and lap counting is integrated it just isn't used due to time restrictions such as current school assignments.
